Dashboard de Escolas em Tempo Integral no Power BI

Publicado em 18 de junho de 2025

Dashboard feito coletando e desenhando manualmente a base de dados das escolas em tempo integral do Espírito Santo.

Power BI Coleta de Dados Visualização de Dados

Este projeto consiste em um dashboard completo, desenvolvido no Microsoft Power BI, para monitorar e analisar os dados das escolas de tempo integral do Espírito Santo.

Um dos principais desafios foi a etapa de coleta e estruturação, que envolveu a criação manual de uma base de dados consolidada a partir de fontes dispersas, garantindo a qualidade e a precisão das informações antes da visualização.
